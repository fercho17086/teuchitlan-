# Teuchitlán, Jalisco - Sitio Web

Bienvenido al sitio web dedicado a **Teuchitlán, Jalisco**, un municipio rico en historia, cultura y tradiciones del occidente de México.

## 📋 Contenido

- **INICIO** - Página principal con información general sobre Teuchitlán
- **HISTORIA** - Zona arqueológica de Guachimontones y su importancia histórica
- **LUGARES TURÍSTICOS** - Principales atractivos y destinos para visitar
- **TRADICIONES** - Celebraciones, costumbres y expresiones culturales locales

## 🏗️ Estructura del Proyecto

```
teuchitlan-/
├── index.html           # Página de inicio
├── historia.html        # Página de historia
├── lugares.html         # Página de lugares turísticos
├── tradiciones.html     # Página de tradiciones
├── css/
│   └── estilos.css      # Hoja de estilos
├── img/                 # Carpeta para imágenes
├── video/               # Carpeta para videos
└── README.md            # Este archivo
```
